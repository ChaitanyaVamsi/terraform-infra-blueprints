# Terraform Infra Blueprints

This repository contains my practice projects using Terraform to build and manage infrastructure as code.

##  Contents

* Terraform basics
* Infrastructure provisioning examples
* Reusable Terraform modules
* Environment-based configurations

##  Tools

* Terraform
* Cloud Infrastructure
* Git & GitHub

##  Getting Started

1. Clone the repository
2. Navigate to a project folder
3. Run the following commands:

```
terraform init
terraform plan
terraform apply
```

##  Purpose

This repository is part of my learning journey to understand and implement Infrastructure as Code using Terraform.
